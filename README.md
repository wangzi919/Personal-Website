# Tzu-Hao Wang Personal Website

This is the personal website of Tzu-Hao Wang, an elegant, modern, and minimalistic personal page inspired by premium blog templates like Ablogia. 

## Design Philosophy

The website embraces simplicity and elegance, combining clean aesthetics with dynamic functionality. It prominently features a high-definition sea otter portrait, reflecting the calming and adaptable nature of these animals.

## Features

*   **Responsive Layout:** Adapts seamlessly to both desktop and mobile devices.
*   **Minimalist UI:** Clean typography using *Inter* for body text and *Playfair Display* for headings.
*   **Dynamic Time Display:** A precise JavaScript time widget that shows the current local time in an elegant, human-readable format, and a floating card accurately reflecting the ISO 8601 formatting based on the user's timezone.
*   **Premium Visuals:** A sleek, minimal color palette featuring off-white, slate gray, and a warm earthy accent color.

## Project Structure

*   `index.html`: The main structural file containing the landing page content.
*   `style.css`: All styling definitions, utilizing CSS variables for easy theming.
*   `script.js`: Handles the dynamic time updates for both the navigation bar and the floating hero card.
*   `sea_otter.png`: The custom-generated minimalist sea otter hero image.

## Getting Started

To view the website locally, simply open the `index.html` file in any modern web browser. 

```bash
# Example if using Chrome
google-chrome index.html
```
